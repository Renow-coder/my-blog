📚 关于这个博客

这个博客采用纯前端方案构建，没有使用任何框架。HTML + CSS + JavaScript， 加上 marked.js 做 Markdown 渲染。每一行代码都清晰可控，每一个细节都经过打磨。

你在这里看到的所有功能——暗黑模式、文章搜索、标签分类、点赞收藏、留言互动、 社交分享——都是用原生 JavaScript 实现的。如果你对实现细节感兴趣， 可以阅读这篇文章。
